# Input Component

The Input component is a reusable form control that supports various input types and states.

## Import

```tsx
import { Input } from '@/components/atoms/Input'
```

## Usage

```tsx
const [value, setValue] = useState('')

<Input
  value={value}
  onChange={setValue}
  placeholder="Enter your text"
/>
```

## Types

The Input component supports various HTML input types:

<div className="space-y-4 mb-6">
  <Input value="" onChange={() => {}} type="text" placeholder="Text input" />
  <Input value="" onChange={() => {}} type="password" placeholder="Password input" />
  <Input value="" onChange={() => {}} type="email" placeholder="Email input" />
  <Input value="" onChange={() => {}} type="number" placeholder="Number input" />
</div>

```tsx
<Input type="text" value={value} onChange={handleChange} />
<Input type="password" value={value} onChange={handleChange} />
<Input type="email" value={value} onChange={handleChange} />
<Input type="number" value={value} onChange={handleChange} />
```

## States

The Input component supports different states:

<div className="space-y-4 mb-6">
  <Input value="Default state" onChange={() => {}} />
  <Input value="Disabled state" onChange={() => {}} disabled />
  <Input value="" onChange={() => {}} placeholder="With placeholder" />
</div>

```tsx
<Input value={value} onChange={handleChange} />
<Input value={value} onChange={handleChange} disabled />
<Input value="" onChange={handleChange} placeholder="With placeholder" />
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | string | required | The current value of the input |
| onChange | (value: string) => void | required | Function called when the input value changes |
| type | 'text' \| 'password' \| 'email' \| 'number' | 'text' | The type of input |
| placeholder | string | undefined | Placeholder text shown when input is empty |
| disabled | boolean | false | Whether the input is disabled |
